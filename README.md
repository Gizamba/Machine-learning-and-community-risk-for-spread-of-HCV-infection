# Machine-learning-and-community-risk-for-spread-of-HCV-infection
Here you will find partial datasets and the R code used in this study.
# Dataset zipped folder
This folder contains the datasets used in the analysis. Data was obtained from the 2018-2022 American Community Survey (ACS) 5-year estimate, AIDSVu Public Data Resource, California Department of Public Health, Substance and Addiction Prevention Branch- California Overdose Surveillance Dashboard, the National Plan and Provider Enumeration System, and the Medi-Cal Managed Care Provider Listing dataset from the California Department of Health Care Services, 2022. Note, some datasets have access restrictions, so they are not included in this folder. 
# R markdown zipped folder
This folder contains the R scripts used to prepare the data and fit the models.
