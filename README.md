# Machine-learning-and-community-risk-for-spread-of-HCV-infection
Here you will find a partial dataset and R code that were used in this study.
# Partial Dataset
Data in this dataset were obtained from the 2018-2022 American Community Survey (ACS) 5-year estimate, AIDSVu Public Data Resource, California Department of Public Health, Substance and Addiction Prevention Branch- California Overdose Surveillance Dashboard, the National Plan and Provider Enumeration System, and the Medi-Cal Managed Care Provider Listing dataset from the California Department of Health Care Services, 2022.
# R code
The R markdown files with the code used to prepare and run the model include Aim_1_data_preps and Model_fitting
